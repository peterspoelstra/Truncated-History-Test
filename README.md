# Truncated History Framework for Synthetic Control Approaches

This repository contains the data and code for the paper  
**“Truncated History Framework for Synthetic Control Approaches”**,  
published in *Economics Letters*.

📄 **Published article**  
https://doi.org/10.1016/j.econlet.2025.112701

---

## Repository contents

The repository is structured around two main R scripts:

- **TH Code Table 1 Abadie et al (2010).R**  
  Reproduces the results in **Table 1**, based on the California tobacco control application.

- **TH Code Table 1_2_3 SDID.R**  
  Reproduces the results in **Tables 1–3**, including the DID, SDID and DIFP-based extensions.

These scripts include the necessary data to replicate the tables reported in the paper.

---

## Additional code

An additional R script is included that applies the proposed **Truncated History robustness check** to alternative applications.  
This script is **not used in the published paper**, but is provided for illustration and further use.
